## ChipAte

first-pass at a Chip-8 emulated 'virtual' computer system in C# and MonoGame

documentation reference:  
https://en.wikipedia.org/wiki/CHIP-8#Opcode_table  
and  
http://devernay.free.fr/hacks/chip8/C8TECH10.HTM
